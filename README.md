## Sobre o Projeto

Este repositório contém o primeiro desafio prático da trilha de **Analista de Power BI**. O objetivo é demonstrar habilidades de modelagem de dados, criação de visuais e construção de relatórios interativos utilizando o Microsoft Power BI.

O projeto consiste em um dashboard desenvolvido no arquivo `Desafio 1.pbix`, acompanhado de uma prévia visual do relatório (`projeto.png`).

## Estrutura do Repositório

```
power_bi_analyst_1/
│
├── Desafio 1.pbix       # Arquivo principal do relatório Power BI
└── projeto.png          # Imagem de prévia do dashboard
```

## Tecnologias Utilizadas

- **Microsoft Power BI Desktop** — Criação do relatório e dashboard
- **DAX (Data Analysis Expressions)** — Medidas e colunas calculadas
- **Power Query (M)** — Transformação e tratamento de dados

## Como Visualizar o Projeto

1. Certifique-se de ter o [Power BI Desktop](https://powerbi.microsoft.com/pt-br/desktop/) instalado em sua máquina.
2. Clone este repositório:
   ```bash
   git clone https://github.com/Gu1lz/power_bi_analyst_1.git
   ```
3. Abra o arquivo `Desafio 1.pbix` no Power BI Desktop.
